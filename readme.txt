-------------------------------------------------------
    Features
-------------------------------------------------------

Tracks includes the following features:

    * Custom logo
    * Tagline display control
    * Social media icons
    * Search icon/input
    * Post meta display controls
    * Comment display controls
    * Custom footer text
    * Custom CSS
    * Custom excerpt length
    * Lazy loading
    * Additional options

Tracks also has a premium upgrades: https://www.competethemes.com/tracks/tracks-theme-upgrades/

-------------------------------------------------------
    Additional Info:
-------------------------------------------------------

    * All menus in Tracks can have 1-3 tiers
    * Featured Images on Posts and Pages have a 2x1 ratio at all screen sizes
    * Tracks is child theme compatible. Download a starter child theme here:
        https://www.competethemes.com/help/child-theme-tracks/
